\chapter{\'Etude du réalisable}
\label{ann:etude-du-realisable}
L'Annexe \ref*{ann:etude-du-realisable} est un extrait de l'étude du réalisable effectuée en début de stage, document sur lequel je me suis appuyé pour définir le cahier des charges.

Sont inclus :

\begin{itemize}
	\item Le sommaire, aperçu rapide du contenu
	\item Un exemple des paramètres à passer pour une requête
	\item Les explications de téléchargement du document en ligne, avec redirection de la sortie standard qui m'a posé problème.
	\item Les informations sur la création d'utilisateur en REST, malheureusement impossible.
	\item Les consignes pour ouvrir un document sans connexion préalable, via une URL (pas de sollicitation REST, mais vitale pour le projet final)
\end{itemize}
